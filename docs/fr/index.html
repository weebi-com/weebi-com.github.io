L'application mobile weebi peut se connecter à tous les types d'imprimantes thermiques Bluetooth. Elle peut aussi être lié à une balance digitale via bluettoth pour accélérer la pesée des produits. Sur mobile la caméro du téléphone fait office de lecteur de code barre, et il est possible de s'intégrer à un lecteur existant sur pc/macos.

La création/mise à jour/suppression d'un produit se fait directement depuis l'application. Il est possible d'importer un catalogue produit depuis un fichier excel. Un modèle du fichier et un tutoriel sont disponibles dans l'application. Il est aussi possible de charger un ancien catalogue de produits weebi en 1 clic.
